## ⚡ Features

# Moderation: 
ban, kick, mute, unmute

## Warnings System: 
warn, warnings, clearwarnings

## Server Control: 
lock, unlock, slowmode

## Info Commands: 
serverinfo, userinfo, avatar, ping

##Events: 
guildMemberAdd, guildMemberRemove, messageDelete, messageUpdate

## Permissions Logs: permmisionchecker, logs

## 🛠️ Requirements

Node.js v18+

Discord.js v14+

(optional) Database e.g., SQLite or MongoDB for warnings)

## 🚀 Setup & Run
1- Install dependencies:
npm install

2- fill the "config.json" with requirements

3- ⚙️ Deploy Slash Commands
       node deploy-commands.js

4- Run The Bot
   node index.js
